## 🏃‍♂️ הרצה מהירה

```bash
# שכפול הפרויקט
git clone <repository-url>
cd task_todo

# הרצה עם Docker
docker compose up -d

# גישה לאפליקציה
http://localhost:5000
```

## 🧪 בדיקות
